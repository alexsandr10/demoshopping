# Demoshopping
Веб-приложение Интернет-магазин (https://qa.demoshopping.ru/). Далее представлена тестовая документация: чек-лист, тест-кейс, баг-репорт. Были применены техники тест-дизайна.

Пароль должен содержать не менее 8 символов, включая минимум одну букву и одну цифру.</li>

В случае, если статус карты valid, а баланс на аккаунте достаточный для покупки, пользователь может совершить покупку. Во всех остальных случаях транзакция будет отклонена.</li>
